Why do you think some version control systems, like Git, allow saving multiple
files in one commit, while others, like Google Docs, treat each file separately?

 Porque o foco do Google Docs é manter o arquivo em uso atualizado, já o Git objetiva o controle de alterações. No Git, tratar as alterações em conjunto é importante, pois os arquivos costumam estar interrelacionados.


How can you use the commands git log and git diff to view the history of files?

  O Git diff permite identificar as alterações, a partir das diferenças entre o arquivo mais recente e o mais antigo. Já o Git log lista as alterações feitas, em ordem cronológica decrescente, com os comentários do autor.
